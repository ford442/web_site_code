# Static HTML Template

A simple, clean starting point for static HTML/CSS/JavaScript websites.

## Structure

```
static-html/
├── index.html         # Main HTML file
├── css/
│   └── style.css     # Stylesheet
├── js/
│   └── main.js       # JavaScript file
├── images/           # Project images
└── README.md         # This file
```

## Getting Started

1. Copy this template to a new project:
   ```bash
   cp -r templates/static-html projects/my-new-site
   cd projects/my-new-site
   ```

2. Update this README.md with your project details

3. Start editing the HTML, CSS, and JavaScript files

4. Open `index.html` in a browser to view your site

## Features

- Clean, semantic HTML5 structure
- Responsive CSS with modern layout
- Vanilla JavaScript (no frameworks required)
- Mobile-friendly viewport settings
- Organized file structure

## Customization

- Edit `index.html` for your content
- Modify `css/style.css` for styling
- Add interactivity in `js/main.js`
- Add images to `images/` directory

## Deployment

This is a static site that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Simply upload the files or connect your repository to the hosting service.

## Browser Support

This template uses modern web standards and supports:
