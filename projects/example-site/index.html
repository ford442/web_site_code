Welcome

This is an example project demonstrating the repository structure.

Projects are organized in separate directories under projects/.

About This Example

This example shows how to:

  • Organize website projects in separate directories
  • Use templates to start new projects
  • Reference shared assets from the assets directory
  • Document your project with a README

Getting Started

To create your own project, copy a template from templates/ to projects/ and customize it!

cp -r templates/static-html projects/my-new-site
